# Restaurant App Client - Project

This is a professional cross-platform mobile application for restaurant management, built as a final project.

##  Key Features
- **User Authentication:** Secure login and registration.
- **Smart Menu:** Dynamic filtering by category, price, and dietary tags.
- **Real-time Cart:** Seamless order management with price calculations.
- **Accessibility Feature:** Integrated **Text-to-Speech** (TTS) for menu item descriptions.
- **Advanced UI:** Dark mode support and full RTL (Right-to-Left) language support for Hebrew.

##  Tech Stack
- **Frontend:** React Native (Expo)
- **State Management:** Context API
- **Navigation:** Expo Router
- **Backend Communication:** Axios (Node.js/Express)

##  How to Run
1. Clone this repository: `git clone https://github.com/Or2699/restaurant-app-client.git`
2. Install dependencies: `npm install`
3. Run the project: `npx expo start`
